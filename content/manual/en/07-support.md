# 7. Help & Support {#support}

Need help? The most useful places are all in this chapter. Check bed status, read help content, or contact the support team — all accessible from the Support area in Settings.

### 7.1 Run Bed Diagnostic {#bed-diagnostic}

Think something's off with the bed? Run a Bed Diagnostic first.

1. On the Settings page, tap Bed Diagnostic.
2. Make sure the area around the bed is clear, then tap Run Bed Diagnostic Test.

![Bed Diagnostic page](/manual/chapter-07/bed-diagnostic/bed-diagnostic-overview.png)

3. Wait for it to finish, then check the results.

![Bed Diagnostic results](/manual/chapter-07/bed-diagnostic/bed-diagnostic-results.png)

### 7.2 Open Help Center {#help-center}

Want to browse help content? Open Help Center.

You'll find User Manual, FAQs, and End User License Agreement — open whatever you need.

![Help Center page](/manual/chapter-07/help-center/help-center-overview.png)

### 7.3 Contact the Support Team {#contact-us}

Need to talk to a person? Open Contact Us.

1. Describe your issue in the input box.
2. Check the email-related options on the page if needed.
3. Tap Send Message to submit.

![Contact Us page](/manual/chapter-07/contact-us/contact-us-form.png)

To help the support team respond faster, include what you saw, when it happened, and what you've already tried.
