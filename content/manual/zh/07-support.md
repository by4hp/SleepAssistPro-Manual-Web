# 7. 获取帮助与联系支持 {#support}

需要帮助时最常用的几个地方都在这里。想检查床体状态、查看说明，或者直接联系支持团队，都可以从 Settings 页面的 Support 区域进入。

### 7.1 运行床体诊断 {#bed-diagnostic}

怀疑床体运行有问题，可以先跑一次 Bed Diagnostic。

1. 在 Settings 页面点 Bed Diagnostic。
2. 确认床体周围安全后，点 Run Bed Diagnostic Test。

![Bed Diagnostic 页面](/manual/chapter-07/bed-diagnostic/bed-diagnostic-overview.png)

3. 等诊断完成，查看结果。

![Bed Diagnostic 结果页](/manual/chapter-07/bed-diagnostic/bed-diagnostic-results.png)

### 7.2 打开帮助中心 {#help-center}

想查看说明内容，打开 Help Center。

里面一般有 User Manual、FAQs 和 End User License Agreement，按需要打开对应页面即可。

![Help Center 页面](/manual/chapter-07/help-center/help-center-overview.png)

### 7.3 联系支持团队 {#contact-us}

需要人工帮助，打开 Contact Us。

1. 在输入框里写下您遇到的问题。
2. 按需要检查页面上的邮箱相关选项。
3. 点 Send Message 提交。

![Contact Us 页面](/manual/chapter-07/contact-us/contact-us-form.png)

为了让支持团队更快理解您的问题，建议写清看到的现象、问题发生的时间，以及已经试过哪些操作。
