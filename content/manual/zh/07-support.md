# 7. 获取帮助与联系支持 {#support}

这一章整理了需要帮助时最常用的几个地方。如果您想先检查床体状态、查看说明，或者直接联系支持团队，都可以从这里开始。相关内容都在 Settings 页面的 Support 区域。

### 7.1 运行床体诊断 {#bed-diagnostic}

如果您怀疑床体运行异常，可以先运行一次 Bed Diagnostic。

1. 在 Settings 页面点击 Bed Diagnostic。
2. 确认床体周围安全后，点击 Run Bed Diagnostic Test。

![Bed Diagnostic 页面](/manual/chapter-07/bed-diagnostic/bed-diagnostic-overview.png)

3. 等待诊断完成，再查看结果。

![Bed Diagnostic 结果页](/manual/chapter-07/bed-diagnostic/bed-diagnostic-results.png)

### 7.2 打开帮助中心 {#help-center}

如果您想查看说明内容，可以打开 Help Center。

这里一般会看到 User Manual、FAQs 和 End User License Agreement。您可以按需要打开对应页面继续查看。

![Help Center 页面](/manual/chapter-07/help-center/help-center-overview.png)

### 7.3 联系支持团队 {#contact-us}

如果您需要人工帮助，可以打开 Contact Us。

1. 在输入框里写下您遇到的问题。
2. 按需要检查页面上的邮箱相关选项。
3. 点击 Send Message 提交。

![Contact Us 页面](/manual/chapter-07/contact-us/contact-us-form.png)

为了让支持团队更快理解您的问题，建议在消息里写清您看到的现象、问题发生的时间，以及您已经试过哪些操作。
