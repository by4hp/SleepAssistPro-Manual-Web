## 1. 快速开始

这一章适合第一次使用 SleepAssistPro 的用户。您只想尽快完成安装、连床和进入首页，也可以先只看这一章。

<a id="download-install"></a>

### 1.1 下载与安装 App

选择您的设备平台，按对应步骤下载并安装。

<a id="download-ios"></a>

#### 1.1.1 在 iOS 下载

打开 iOS App Store，搜索 SleepAssist Pro，然后下载并安装。正式发布后，您也可以通过官方下载链接进入下载页面。

**iOS**: 在 App Store 下载

![iOS](../public/manual/chapter-01/app-store.png)

适用于 iPhone 和 iPad，下载后按提示完成安装。

<a id="download-android"></a>

#### 1.1.2 在 Android 下载

打开 Google Play，搜索 SleepAssist Pro，然后下载并安装。正式发布后，您也可以通过官方下载链接进入下载页面。

**Android**: 在 Google Play 下载

![Android](../public/manual/chapter-01/google-play.png)

适用于 Android 设备，下载后按提示完成安装。

<a id="first-use-steps"></a>

### 1.2 首次使用会经过哪些步骤

第一次使用时，您通常会依次经过下面几个阶段：

1. 下载并打开 App
2. 连接并识别您的床
3. 如果连接的是智能床，注册或登录账号
4. 完成床体设置和首次资料填写
5. 进入首页，开始日常使用

如果您现在已经装好 App，准备开始连床，直接继续看 [2.2 开始连接床](#start-connection) 即可。

<a id="first-connection"></a>

## 2. 第一次连接与设置

这一章适合第一次连接床的用户。您按顺序往下操作，就能完成识别、连接和首次设置。

<a id="preparation"></a>

### 2.1 开始前准备

开始连床前，先把这些准备项看一遍。先确认电源、权限、网络和设备信息，后面会省事很多。

开始前，先检查下面几项：

- 床体已通电，并处于可连接状态
- 手机已靠近床体，方便 App 搜索附近设备
- 手机可正常使用蓝牙、位置和相机权限
- 如果要连接 Wi-Fi 型智能床，请提前确认可用的 Wi-Fi 网络和密码
- 如果自动识别失败，您可能需要扫描床体二维码，或手动输入设备号

确认这些都没问题后，再继续看 [2.2 开始连接床](#start-connection)。

<a id="start-connection"></a>

### 2.2 开始连接床

前一节准备好后，就可以从欢迎页开始连床。

第一次打开 App 时，您会先看到欢迎页。这里有两个主要按钮：Connect Your Bed 和 No Bed Yet?

1. 如果您已经有床，点击 Connect Your Bed。
2. 如果您还没有床，点击 No Bed Yet?，先前往品牌官网了解产品或购买床体。

![欢迎页](../public/manual/chapter-02/onboarding/01-Entry-And-Setup/entry-welcome.png)

如果您已经开始连床，下一步继续看 [2.3 查找并识别您的床](#find-identify-bed)。

<a id="find-identify-bed"></a>

### 2.3 查找并识别您的床

从欢迎页进入后，先找到床，再确认床型。先试自动识别；找不到时，再扫码或手动输入设备号。确认床型后，再继续看 [2.4 按床型继续设置](#bed-type-setup)。

<a id="auto-identify"></a>

#### 2.3.1 自动识别

这是最推荐的方式。大多数用户都可以在这里完成识别。

1. 点击 Connect 开始找床。

![开始找床](../public/manual/chapter-02/onboarding/01-Entry-And-Setup/entry-bed-setup-instructions.png)

2. 如果系统请求蓝牙权限，请点击允许。否则 App 无法搜索附近可连接的床。

![蓝牙权限提示](../public/manual/chapter-02/onboarding/02-Bed-Search-And-Identification/bed-search-bluetooth-permission-required.png)

3. 等待 App 自动搜索并识别附近可连接的床。

![搜索中](../public/manual/chapter-02/onboarding/02-Bed-Search-And-Identification/bed-search-loading.png)

4. 如果页面显示设备列表，请选择您的床并继续。

![搜索结果](../public/manual/chapter-02/onboarding/02-Bed-Search-And-Identification/bed-search-results-multiple-beds.png)

5. 如果 App 找到了设备，但还没有识别出床型，先选择您的设备，再按页面提示进入床型选择页。

![设备已找到但未识别床型](../public/manual/chapter-02/onboarding/02-Bed-Search-And-Identification/bed-selection-before-type-identification.png)

6. 在床型选择页中确认正确的床型，然后继续。

![床型选择](../public/manual/chapter-02/onboarding/02-Bed-Search-And-Identification/bed-identification-type-selection.png)

<a id="qr-identify"></a>

#### 2.3.2 扫码识别

如果自动识别没有找到您的床，请点击 Try another way。

1. 在 Unable to Connect 页面点击 Scan QR Code with Camera。

![无法连接页](../public/manual/chapter-02/onboarding/03-Alternative-Connection/alternative-connection-unable-to-connect.png)

2. 如果系统请求相机权限，请点击允许。

![相机权限提示](../public/manual/chapter-02/onboarding/03-Alternative-Connection/qr-scan-camera-permission-required.png)

3. 将摄像头对准床体上的二维码进行扫描。

![扫码页](../public/manual/chapter-02/onboarding/03-Alternative-Connection/qr-scan-overview.png)

4. 连接成功后，按页面提示继续下一步。

<a id="manual-identify"></a>

#### 2.3.3 手动输入设备号

如果扫码不可用，或者二维码识别失败，您可以改用手动输入设备号。

1. 在 Unable to Connect 页面点击 Manually Enter Bluetooth ID，并根据图示找到设备号。

![无法连接页](../public/manual/chapter-02/onboarding/03-Alternative-Connection/alternative-connection-unable-to-connect.png)

2. 在输入页填写床体上的设备号。

![蓝牙 ID 输入页](../public/manual/chapter-02/onboarding/03-Alternative-Connection/manual-bluetooth-id-entry.png)

3. 确认输入无误后，继续下一步。如果输入的设备号格式不正确，页面会直接提示错误。修改后再继续即可。

![输入错误提示](../public/manual/chapter-02/onboarding/03-Alternative-Connection/manual-bluetooth-id-error.png)

4. 识别成功后，按页面提示继续连接。

<a id="skip-connect"></a>

#### 2.3.4 还不想连接床

如果您现在不想连接床，可以先点击 Skip To Sign In。

![连床说明页](../public/manual/chapter-02/onboarding/01-Entry-And-Setup/entry-bed-setup-instructions.png)

如果您是第一次连床，或者这张床还没有绑定到您的账号，请继续看前面几个小节。

<a id="bed-type-setup"></a>

### 2.4 按床型继续设置

确认床型后，接下来会分成两种情况。电动床可以直接进入首页；智能床还要注册或登录账号、连接床体，再填写第一次使用的信息。

<a id="electric-bed-setup"></a>

#### 2.4.1 电动床

识别到电动床后，App 会直接进入 Homepage。电动床不需要登录账号，到这里第 2 章就基本完成了。接下来您可以直接看第 3 章的 [电动床首页](#homepage-electric) 和 [遥控器](#remote)。

![电动床首页](../public/manual/chapter-02/homepage/electric-bed-homepage.png)

<a id="smart-bed-setup"></a>

#### 2.4.2 智能床

识别到智能床后，如果您是第一次使用，请继续完成注册、床体连接和首次资料填写。下一步先看 [2.5 首次注册智能床账号](#register-account)。

<a id="register-account"></a>

### 2.5 首次注册智能床账号

这一节写给第一次用智能床、还没有账号的用户。您可以用邮箱、Apple 或 Google 注册。注册并验证后，再继续看 [2.6 设置智能床](#setup-smart-bed)。

<a id="register-email"></a>

#### 2.5.1 使用邮箱注册

如果您打算用邮箱注册，请直接在 Sign Up 页面完成。

1. 输入邮箱地址。

![注册空白页](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/sign-up-empty-form.png)

2. 设置密码并再次输入密码进行确认。

![邮箱与密码填写页](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/sign-up-email-entered.png)

3. 选择您所在的地区。

![地区选择页](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/sign-up-region-selection.png)

4. 勾选必要的协议确认项。

![协议勾选状态](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/sign-up-agreement-checked.png)

5. 点击注册按钮。

![注册提交页](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/sign-up-submit-clean-shot.png)

6. 查看系统发送到邮箱里的验证码，并输入验证码并完成邮箱验证。

![邮箱验证码页](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/email-verification-clean-shot.png)

<a id="register-apple"></a>

#### 2.5.2 使用 Apple 注册

如果您已经在手机上登录了 Apple 账号，可以直接点击 Apple 入口继续注册。

1. 在注册页面底部点击 Apple。

![Apple 入口](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/sign-up-submit-clean-shot.png)

2. 按系统页面完成 Apple 授权。

![Apple 授权页](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/sign-up-apple-authorization.png)

3. 完成账号创建后，回到 App 继续后面的设置。

<a id="register-google"></a>

#### 2.5.3 使用 Google 注册

如果您已经在手机上登录了 Google 账号，可以直接点击 Google 入口继续注册。

1. 在注册页面底部点击 Google。

![注册页](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/sign-up-empty-form.png)

2. 按系统页面完成 Google 授权。

![Google 授权页](../public/manual/chapter-02/onboarding/04-Account-Registration-And-Verification/sign-in-google-authorization-placeholder.svg)

3. 完成账号创建后，回到 App 继续后面的设置。

<a id="setup-smart-bed"></a>

### 2.6 设置智能床

账号准备好后，接着把床连好。蓝牙型和 Wi-Fi 型的设置步骤不一样，连好以后还要确认是否打开睡眠数据跟踪。做完这里，再继续看 [2.7 完成首次资料填写](#first-profile)。

<a id="setup-bt-bed"></a>

#### 2.6.1 设置蓝牙型智能床

如果识别到的是蓝牙型智能床，操作会简单一些。

1. 等待 App 完成蓝牙连接。

![蓝牙连接阶段](../public/manual/chapter-02/onboarding/05-Smart-Bed-Connection/bluetooth-connection-stage-bluetooth.png)

2. 等待 App 完成床体设置。

![床体设置阶段](../public/manual/chapter-02/onboarding/05-Smart-Bed-Connection/bluetooth-connection-stage-bed-setup.png)

3. 连接成功后，进入睡眠追踪设置。

![蓝牙连接完成](../public/manual/chapter-02/onboarding/05-Smart-Bed-Connection/bluetooth-connection-complete.png)

<a id="setup-wifi-bed"></a>

#### 2.6.2 设置 Wi-Fi 型智能床

如果识别到的是 Wi-Fi 型智能床，您还要连上家里的 Wi-Fi。

1. 选择可用的 Wi-Fi 网络。

![Wi-Fi 选择页](../public/manual/chapter-02/onboarding/05-Smart-Bed-Connection/wifi-network-selection.png)

2. 输入 Wi-Fi 密码并继续。

![Wi-Fi 密码输入页](../public/manual/chapter-02/onboarding/05-Smart-Bed-Connection/wifi-password-entry-empty.png)

3. 等待 App 完成蓝牙连接。

![蓝牙连接阶段](../public/manual/chapter-02/onboarding/05-Smart-Bed-Connection/wifi-connection-stage-bluetooth.png)

4. 等待 App 完成 Wi-Fi 连接。

![Wi-Fi 连接阶段](../public/manual/chapter-02/onboarding/05-Smart-Bed-Connection/wifi-connection-stage-wifi.png)

5. 等待 App 完成服务器连接。

![服务器连接阶段](../public/manual/chapter-02/onboarding/05-Smart-Bed-Connection/wifi-connection-stage-server.png)

6. 连接成功后，进入睡眠追踪设置。

![Wi-Fi 连接完成](../public/manual/chapter-02/onboarding/05-Smart-Bed-Connection/wifi-connection-complete.png)

<a id="setup-sleep-tracking"></a>

#### 2.6.3 开启睡眠数据跟踪

蓝牙型或 Wi-Fi 型智能床连好后，页面会显示 Sleep Data Tracking。

1. 先阅读页面说明。

![睡眠数据跟踪说明页](../public/manual/chapter-02/onboarding/06-Sleep-Data-Tracking/sleep-data-tracking-overview.png)

2. 选择是否开启睡眠数据跟踪。如果您关闭睡眠数据跟踪，App 将不再追踪您的睡眠数据，但同时也不会生成睡眠报告。您之后如果需要，也可以在设置中重新打开。

![关闭确认提示](../public/manual/chapter-02/onboarding/06-Sleep-Data-Tracking/sleep-data-tracking-turn-off-confirmation.png)

3. 点击继续，进入首次资料填写。

![昵称输入页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/01-Shared-Steps/profile-name-entry-empty.png)

<a id="first-profile"></a>

### 2.7 完成首次资料填写

床连好并确认睡眠数据跟踪后，还要补充第一次使用的信息。您需要填写昵称、基础信息、睡眠目标和个人偏好；如果床支持分区功能，页面还会多一步床边位置选择。完成这里后，您就可以进入首页，开始看 [3.2 智能床首页](#homepage-smart)。

<a id="profile-no-zone"></a>

#### 2.7.1 无分区功能智能床

1. 输入昵称。

![昵称填写页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/01-Shared-Steps/profile-name-entry-filled.png)

2. 阅读页面上的填写说明。

![资料填写说明页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/01-Shared-Steps/profile-setup-introduction.png)

3. 进入基础信息填写页，补充性别、身高、体重、床垫厚度、生日和时间制式等内容。

5. 继续进入个人标签页，选择适合自己的标签。您也可以点击 Skip 先跳过。

![个人标签页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/02-Without-Partition-Feature/sleep-preferences-selection-state.png)

6. 完成后进入首页。

<a id="profile-zone"></a>

#### 2.7.2 有分区功能智能床

1. 输入昵称。

![昵称填写页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/01-Shared-Steps/profile-name-entry-filled.png)

2. 阅读页面上的填写说明。

![资料填写说明页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/01-Shared-Steps/profile-setup-introduction.png)

3. 先选择您的床边位置。

4. 进入基础信息填写页，并先确认单位设置。

![单位选择页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/03-With-Partition-Feature/sleep-profile-units-selection.png)

![单位配置页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/03-With-Partition-Feature/sleep-profile-units-configuring.png)

5. 完成基础信息后，继续设置睡眠目标，并选择优先改善的睡眠项。

![睡眠目标优先项页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/03-With-Partition-Feature/sleep-goals-priority-selection.png)

6. 进入个人标签完成页，确认后点击 Finish 进入首页。

![完成页](../public/manual/chapter-02/onboarding/07-Profile-And-Preferences/03-With-Partition-Feature/sleep-preferences-final-state.png)

看到首页后，第一次连接和基础设置就完成了。

## 3. 使用 App

这一章适合已经完成连床和首次设置的用户。您可以从这里开始熟悉首页、报告、实时数据、遥控器和睡眠计划这些常用功能。

### 3.1 登录

1. 在 Sign In 页面底部点击 Apple。

![Apple 登录入口](../public/manual/chapter-03/login/Pasted%20image%2020260325161306.png)

## 4. AI 对话

这一章写给智能床用户。AI Chat 是 App 里的文字助手。它会帮您了解床体功能、查看功能怎么用、处理基础问题，也会在合适的时候带您打开对应页面。

<a id="ai-capabilities"></a>

### 4.1 AI 现在可以帮您做什么

您可以把 AI Chat 当成一个随时可用的智能床助手。当前版本更适合帮您处理下面几类问题：

- 问床体功能是什么，例如某个模式、预设或传感功能是什么意思
- 问某个功能怎么用，例如怎么解锁、怎么设置、怎么进入某个功能
- 处理基础故障，例如床体无法移动、遥控器没有响应这类常见问题
- 识别明确的控制意图，并把您带到正确的控制入口
- 当您对睡眠数据有疑问时，解释基础数据来源，并引导您提交反馈

如果问题超出当前支持范围，AI Chat 也可能把您引导到人工客服或反馈表单。

<a id="enter-ai-chat"></a>

### 4.2 进入 AI 对话

首页和指标详情页都会出现 AI 入口。

1. 在首页点击底部的 Ask me。

![首页 Ask me 入口](../public/manual/chapter-04/ai-chat/ai-chat-entry-homepage-ask-me.png)

2. 在指标详情页的日报中点击 Ask me。

![指标详情页 Ask me 入口](../public/manual/chapter-04/ai-chat/ai-chat-entry-metric-detail-ask-me.png)

3. 进入页面后，查看推荐问题或当前已有建议。

<a id="ai-suggestions"></a>

### 4.3 查看 AI 建议并继续提问

进入 AI Chat 后，您可以先看系统给出的建议，再决定是否继续提问。提问时，尽量直接说出您现在想做什么，或者眼前遇到了什么问题。

1. 先阅读当前页面展示的 AI 建议。
2. 如果页面提供推荐问题，直接点击就可以查看回复。
3. 如果您想继续提问，直接输入问题即可。
4. 如果您不知道怎么描述，可以直接写出功能名、问题现象，或您想执行的动作。

![AI 建议与输入页](../public/manual/chapter-04/ai-chat/ai-chat-suggestions-overview.png)

您可以直接这样问：

- What is Auto Anti-Snore Intervention?
- How do I unlock Child Lock?
- Why does my base not respond to my remote?
- Go to zero gravity.
- Why is my sleep duration incorrect?

<a id="ai-help-process"></a>

### 4.4 AI 会怎么帮助您

根据您输入的问题不同，AI Chat 的处理方式也会不一样。您通常会看到下面几种结果：

- 如果您问的是怎么操作，AI 会按步骤告诉您下一步怎么做。

![AI 操作指引示例](../public/manual/chapter-04/ai-chat/ai-chat-help-how-to.png)

- 如果您说的是查看某个指标或是询问相关信息，AI 会列出相应的图表数据。

![AI 图表数据示例](../public/manual/chapter-04/ai-chat/ai-chat-help-data-chart.png)

- 如果您表达的是明确操作意图，例如想让床进入某个预设，AI 可能会显示对应功能卡片，并引导您进入 Remote 页面操作。

![AI 功能卡片示例](../public/manual/chapter-04/ai-chat/ai-chat-help-remote-card.png)

<a id="human-support"></a>

### 4.5 什么时候需要人工支持

AI Chat 可以帮助您处理常见问题，但它不是完整售后诊断工具。如果问题没有解决，页面可能会继续给您人工支持入口或反馈入口。

您更适合改用人工支持或反馈表单的情况包括：

- 按 AI 给出的基础步骤操作后，问题仍然没有解决
- 床体持续无法连接、无法移动，或重复出现同一异常
- 您需要提交具体的睡眠数据问题，帮助后续排查和校准

<a id="ai-issues"></a>

### 4.6 遇到问题时

如果 AI Chat 只显示基础提示，或者您觉得回复和当前情况不一致，请先检查下面几项：

- 当前是否已经连接智能床
- 当前页面是否有可用数据
- 您输入的问题里是否已经说清了功能名、问题现象或目标动作

如果 AI Chat 仍然没有给出可用结果，请先回到 [8.6 AI 内容或实时数据暂不可用](#ai-data-unavailable)，再决定是否改用人工支持或反馈表单。

<a id="bed-side-settings"></a>

#### 5.2.2 调整床侧

如果您睡的位置和当前设置不一致，可以在这里改床侧。改对之后，后面和床侧相关的设置与数据展示会更容易对应到您自己。

1. 点击 Bed Side。
2. 在弹出的 Bed Side 面板里选择 Center、Left 或 Right。
3. 点击 Confirm 保存。

<a id="anti-snore-settings"></a>

#### 5.2.3 调整防打鼾灵敏度

这一项用来控制防打鼾检测是否开启，以及触发时有多敏感。如果您希望床更容易识别到打鼾并做出反应，可以选更高的灵敏度；如果不想用这项功能，也可以直接关闭检测。

1. 点击 Anti-Snore Sensitivity。
2. 通过 Detect Snoring 开关决定是否开启防打鼾检测。
3. 如果保持开启，再选择 Low、Medium 或 High。

![防打鼾灵敏度](../public/manual/chapter-05/bed-settings/bed-settings-anti-snore-sensitivity.png)

<a id="sleep-tracking-settings"></a>

#### 5.2.4 控制睡眠数据跟踪

这一项决定床体后面的睡眠数据还要不要继续同步到 App。关闭后，您仍然可以看以前已经有的数据，但新的睡眠数据不会继续进来。

1. 留在 Bed Settings 页面。
2. 直接打开或关闭 Sleep Data Tracking 开关。

关闭后，App 不会继续记录新的床体睡眠数据；重新打开后，之后的新数据才会继续写入。

<a id="disconnect-bed"></a>

#### 5.2.5 断开当前床体

如果您不再使用当前这张床，或者准备换连另一张床，可以在这里断开。断开后，当前账号通常还能查看历史数据，但在重新连接之前，不会继续接收新的床体数据。

1. 点击 Disconnect Bed。
2. 在确认弹窗里再次阅读提示。
3. 确认无误后，点击 Disconnect。

![断开床体确认弹窗](../public/manual/chapter-05/bed-settings/bed-settings-disconnect-dialog.png)

<a id="hardware-update"></a>

#### 5.2.6 安装硬件更新

如果这里出现 New，说明床体有新的硬件版本可装。更新通常是为了修复问题或补充能力；开始更新后，建议先留在床附近，等进度完成再离开。

1. 点击 Hardware Update。

![Hardware Update 入口](../public/manual/chapter-05/bed-settings/bed-settings-hardware-update-entry.png)

2. 在 Bed Update 弹窗中点击 Update Now。

3. 按页面提示留在床附近，等待更新进度完成。

![硬件更新进度](../public/manual/chapter-05/bed-settings/bed-settings-hardware-update-progress.png)

4. 如果更新失败，您可以点击 Try Again 重新开始，或点击 Later 稍后再处理。

![硬件更新失败弹窗](../public/manual/chapter-05/bed-settings/bed-settings-hardware-update-failed-dialog.png)

#### 5.3.1 设置目标睡眠时长

1. 在 Settings 页面点击 Sleep Goal。
2. 在 Sleep Goal 里选择您希望每晚睡多久。如果您要调整目标睡眠时长，通常会看到下面这个时间选择面板。

<a id="sleep-goal-type"></a>

#### 5.3.2 设置作息类型和改善重点

如果您想让 App 更了解您的作息习惯和当前最在意的问题，可以在这里补充作息类型和改善重点。这些内容更偏向个性化设置，不需要一次想得特别复杂，先选最接近自己的就可以。

1. 在作息类型里选择最接近自己的类型，例如 Lion、Bear 或 Wolf。
2. 在改善目标里选择您当前最想优先改善的一项。
3. 点击 Confirm 保存。

![Sleep Goal 总览](../public/manual/chapter-05/sleep-goal/sleep-goals-overview.png)

## 8. 常见问题与排查

这一章整理了最常见的问题。如果您在连接、登录、查看报告或使用 AI 时遇到问题，可以直接按眼前的情况查找对应小节。先看最接近您当前情况的标题，通常会更容易找到能直接照着做的办法。

<a id="connection-issues"></a>

### 8.1 搜索不到床或连接失败

如果 App 搜索不到床，或者连床时卡住不往下走，请按下面的顺序检查：

<a id="check-connection-conditions"></a>

#### 8.1.1 先检查连接条件

1. 确认床体已经通电，并且手机靠近床体。
2. 重新进入找床流程，再次搜索。

<a id="auto-identify-failed"></a>

#### 8.1.2 自动识别失败时

1. 如果自动识别失败，改用 Scan QR Code with Camera 扫码识别。
2. 如果扫码仍不可用，再改用手动输入设备号。
3. 如果页面进入床型选择页，请手动确认正确的床型后继续。

如果还是连不上，再继续看 [8.2 权限未开启怎么办](#permissions)。

<a id="permissions"></a>

### 8.2 权限未开启怎么办

第一次连床时，App 可能会请求蓝牙、位置和相机权限。

- 蓝牙权限会影响设备搜索和连接。
- 位置权限会影响附近设备扫描。
- 相机权限会影响二维码扫描。

如果您点了“不允许”，页面可能会卡在搜索、扫码或识别。请先打开手机系统设置，给 App 补上权限，再回到 App 重试。

<a id="wifi-connection"></a>

### 8.3 Wi-Fi 型智能床连接失败

如果 Wi-Fi 型智能床卡在 Wi-Fi 或服务器连接这一步，请按下面的顺序处理：

1. 确认所选 Wi-Fi 网络正确。
2. 确认输入的 Wi-Fi 密码无误。
3. 让手机保持在床体附近，等待 App 完成蓝牙、Wi-Fi 和服务器连接。
4. 如果页面长时间没有变化，返回上一步，重新选择 Wi-Fi 后再次尝试。
5. 如果仍然失败，请重新开始找床和设置。

<a id="disconnected"></a>

### 8.4 首页显示未连接

如果首页显示未连接，说明 App 当前没有连上床体。您可以按下面的步骤处理：

1. 先确认页面上出现了未连接提示和 Proceed to bed connection 入口。
2. 点击 Proceed to bed connection。
3. 按页面提示重新连接您的床。
4. 如果重新连接时又遇到搜索或识别问题，再回看 [8.1 搜索不到床或连接失败](#connection-issues) 和 [8.2 权限未开启怎么办](#permissions)。

![首页未连接提示](../public/manual/chapter-08/homepage/No%20connection.png)

<a id="no-report"></a>

### 8.5 首页显示 No Report

如果首页显示 No Report，说明当前日期还没有可显示的睡眠报告。这个提示只会出现在智能床。

1. 先确认首页上显示的是 No Report。
2. 点击日期切换到其他日期，查看是否有可用数据。
3. 确认账号和床体已正常连接。
4. 如果当前日期刚结束一次新的睡眠，请等待数据生成后再返回查看。

![首页 No Report 提示](../public/manual/chapter-08/homepage/No%20report.png)

如果一直看不到数据，请再检查一次床体连接状态。

<a id="ai-data-unavailable"></a>

### 8.6 AI 内容或实时数据暂不可用

如果 AI Chat 只显示基础提示，或者实时数据页没有数值，请先检查下面这几项：

- 当前日期没有足够的睡眠数据。
- App 当前没有与床体保持正常连接。
- 相关内容还在刷新。

先确认首页已经连上床体，所选日期也有有效数据，然后重新进入相关页面。

<a id="login-issues"></a>

### 8.7 收不到验证码或无法完成登录

如果注册、登录或重置密码时卡住了，请先检查这几项：

1. 确认输入的邮箱地址是否正确。
2. 等待系统邮件发送完成后，再查看邮箱。
3. 如果是登录已有账号，请确认当前使用的是正确的登录方式，例如邮箱登录、Apple 登录或 Google 登录。
4. 如果您无法继续登录，请在 Sign In 页面点击 Forgot Password?，然后按邮件提示重新设置密码。
